# 护眼提醒中文使用说明

![界面预览](assets/eye-breaker-preview.png)

## 这是什么

`护眼提醒` 是一个 macOS 本地护眼休息提醒工具。它会按你设置的工作时长提醒你休息，并提供全屏休息浮窗、延后、跳过和 `Esc` 快捷键。

## 快速下载

如果你只是想先试用，不用先打开源码页，直接下载发布版：

- [EyeBreaker.zip](https://github.com/yadongz097-create/eye-breaker/releases/download/v1.0.0/EyeBreaker.zip)

下载后解压，把 `护眼提醒.app` 拖到 `Applications`，然后双击打开。

## 怎么安装

如果你已经下载了发布版，直接把 `护眼提醒.app` 拖到 `Applications` 即可。

如果你想自己编译：

```bash
swift build -c release
./scripts/package-app.sh release
```

打包产物在：

```bash
dist/护眼提醒.app
```

## 怎么使用

1. 打开 `护眼提醒`。
2. 在设置窗口里填写 `工作分钟` 和 `休息分钟`。
3. 需要开机自动启动时，勾选 `开机自启`。
4. 点击 `保存`。
5. 从这一步开始，程序会按新的设置重新计时。

## 设置项说明

- `工作分钟`：连续工作多久后提醒一次。
- `休息分钟`：每次休息持续多久。
- `开机自启`：登录 macOS 后自动打开应用。
- `常用工作时长`：20、25、30、45 分钟快捷按钮。
- `常用休息时长`：1、3、5 分钟快捷按钮。

注意：

- 工作分钟和休息分钟都至少是 `1`。
- 你修改设置后，点 `保存` 会立即生效并重置当前计时。
- 如果你直接关闭设置窗口，只要内容改过，也会自动保存并重置。
- `取消` 只负责关闭窗口，不保存修改。

## 休息浮窗怎么操作

休息时间到了以后，会出现全屏浮窗，常见操作有：

- `延后 5 分钟`：把本次休息往后推迟 5 分钟。
- `跳过本次休息`：直接跳过这次休息。
- `Esc`：等同于“跳过本次休息”。

## 常见问题

### 为什么我改了分钟数，但感觉没生效

先确认有没有点 `保存`。

如果你是计时已经开始以后再改的，保存后会按新设置继续，但当前这一轮会立刻重置，不需要手动重启。

### 关闭设置窗口会不会保存

会。只要你改过内容，直接关闭窗口也会保存。

### 我想退出

直接在菜单栏或 Dock 里退出 `护眼提醒` 即可。退出后不会继续提醒。
